# 贡献指南

感谢您对本项目的关注！我们欢迎各种形式的贡献。

## 🤝 如何贡献

### 报告问题

如果您发现了bug或有功能建议，请：
1. 检查[Issues](https://github.com/your-username/gpt2-transformer/issues)中是否已有相关问题
2. 如果没有，请创建新的Issue，详细描述问题或建议

### 提交代码

1. **Fork本项目**
2. **创建功能分支**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **进行修改**
   - 遵循现有的代码风格
   - 添加必要的注释
   - 确保代码可以正常运行

4. **提交更改**
   ```bash
   git commit -m "Add: 描述你的更改"
   ```

5. **推送到分支**
   ```bash
   git push origin feature/your-feature-name
   ```

6. **创建Pull Request**
   - 详细描述你的更改
   - 说明为什么需要这个更改
   - 如果有相关Issue，请引用

## 📝 代码规范

- 使用有意义的变量和函数名
- 添加必要的注释（中文）
- 保持代码格式一致
- 遵循现有的模块化结构
